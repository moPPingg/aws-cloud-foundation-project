Project Overview

Hands-on AWS foundations with practical cloud workflow exposure

This project is a hands-on AWS foundation lab built to strengthen cloud deployment skills, cost awareness, Linux server practice, and basic monitoring. It is designed as a beginner portfolio project that shows how core AWS services can support a simple but realistic learning path from local development to cloud-hosted delivery.

Labs

Core practice areas in this learning project

Lab 01

Amazon S3 Static Website

Hosting a simple static portfolio site on Amazon S3 to understand cloud storage and public web delivery.

Lab 02

AWS CLI Deployment

Using the AWS CLI to upload and manage website files as part of a repeatable deployment workflow.

Lab 03

AWS Budgets & Cost Monitoring

Setting up budget awareness practices to better understand cloud usage and avoid unnecessary spending.

Lab 04

EC2 Linux Practice

Connecting to a Linux instance for command-line practice, server familiarity, and basic cloud operations.

Lab 05

CloudWatch Basic Monitoring

Exploring entry-level monitoring concepts such as system visibility and simple alerts for instance health.

Architecture Overview

Simple text-based service flow

Developer Laptop -> AWS CLI -> Amazon S3 -> User Browser

Developer Laptop -> SSH -> EC2 Ubuntu Instance -> CloudWatch CPU Alarm

AWS Account -> AWS Budgets -> Email Alert
